### Set up project
1. Install [Docker](https://docs.docker.com/get-docker/)
2. Pull docker image:
```shell
docker build -t my-spark-img .
```
3. Run image:
```shell
build -t my-spark-img .
```

PS: PySpark will be also defined in the `requirements.txt` file.
